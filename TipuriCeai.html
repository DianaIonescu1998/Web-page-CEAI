Tipuri de ceai

Camellia sinensis este singura planta din ale carei frunze, prin diferite procedee, se obtin mai multe tipuri de ceai.

Cu toate ca, denumirea de ceai defineste aceasta planta, ea este folosita generic pentru toate bauturile obtinute din fructe sau frunze uscate.

Ceaiul din diferite plante medicinale precum menta, flori de tei, sunatoarea sunt consumate pentru efectele lor benefice pentru sanatate.

Ceaiuri obtinute din Camellia sinensis

Ceaiul negru

Este obtinut din fruzele arborelui Camellia sinensis

  • frunzele sunt lasate la oxidat intre 2 si 4 saptamani;
  • urmeaza procesul de vestejire si rulare;
  • dupa ce capata o culoare brun- aramie, sunt depozitae in uscatoare speciale;
  • procesul se incheie cand, acestea, sunt sortate in functie de gradul de oxidare.

Ceaiul alb

Este considerat a fi cel mai pretios ceai si este produs in cantitati mici.

Provenit din acelasi arbore precum ceaiul negru, ceaiul alb contine doar frunzele tinere, acoperite cu un praf argintiu. Procesul de preparare este minim.

Ceaiul verde

Este obtinut din frunze ajunse la maturitate.

  • frunzele sunt trecute prin abur sau torefiate;
  • urmatorul pas in procesarea lor este uscarea si rularea, evitandu-se astfel fermentarea.

Ceaiul oolong

Este un ceai aromat.

Este un produs semi- fermentat.

Ceaiul galben

Este procesat similar cu Ceaiul verde, dar uscarea frunzelor este mai lenta.

Are o culoare galben- verzuie si mirosul seamana cu cel al Ceaiului negru

Ceaiul pu-erh

Frunzele sanatoase sunt culese din copaci batrani de ceai.

Procesul de oxidare al acestora este oprit prin prajire.

Apoi sunt rulate, fara sa se rupa, si lasate la uscat pana cand nivelul de umiditate este maxim fara ca acestea sa se sfarame.

In urma acestui proces se obtin doua tipuri:

  1. sheng (crud) - fiind supuse proceselor de aburire si de oxidare lent
  2. shu - fiind supuse procesului de oxidare fortata, apoi sunt tratate cu abur fierbinte si presate in forme
